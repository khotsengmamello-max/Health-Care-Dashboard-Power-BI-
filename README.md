An interactive healthcare analytics dashboard built with **Microsoft Power BI** to help hospital management teams monitor key performance metrics, optimize patient flow, and make data-driven decisions.

## 🎯 Objective

The primary goal of this project is to provide an interactive and insightful dashboard for **Hospital Management Teams** to track and analyze:

- Patient visit trends
- Instrument availability
- Revenue generation across different age groups
- Appointment waiting times
- Patient feedback on medical prescriptions and hospital services

---

## 🛠️ Project Overview

| Component | Description |
|-----------|-------------|
| **Data Source** | MS Excel |
| **Data Transformation** | Used Power Query Editor for merging, trimming, removing duplicates, and changing column data types to ensure data consistency and accuracy |
| **Data Modeling** | Designed an efficient data model establishing relationships between fact and dimension tables (bed details, staff details, department, and dataset tables) |
| **Visualizations** | Developed KPIs using card visuals, bar charts, line charts, maps, and tables |
| **Custom Visuals** | Enhanced user experience with rotating cards, scrollers, and other custom visuals |
| **Interactivity** | Implemented drill-down and drill-through capabilities for detailed patient views |
| **Navigation** | Added filters, slicers, and button navigators for seamless report browsing |

---

## ✅ Key Features

- 📈 **Interactive Visualizations** – Bar charts, line charts, maps, and tables
- 🎯 **KPI Tracking** – At-a-glance performance metrics using card visuals
- 🔍 **Drill-Down Capabilities** – Explore patient-level details with drill-through functionality
- 🧩 **Custom Visuals** – Enhanced UI with rotating cards and scrollers
- 🧭 **User-Friendly Navigation** – Buttons, slicers, and filters for easy data segmentation

---

## 💡 Benefits

- **Centralized Information** – Single source of truth for monitoring hospital activities
- **Data-Driven Decision Making** – Empowers management teams to make informed, evidence-based decisions
- **Improved Operational Efficiency** – Identify bottlenecks and optimize resource allocation

## 🚀 Getting Started

1. Clone this repository
2. Open the `.pbix` file in **Power BI Desktop**
3. Ensure the data source connections are correctly mapped
4. Explore the interactive dashboards
